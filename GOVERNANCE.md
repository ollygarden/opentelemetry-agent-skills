# Governance

OpenTelemetry Agent Skills is an open source project sponsored and maintained by
[OllyGarden](https://ollygarden.com/).

## Roles

- **Contributors** propose changes through issues and pull requests. Contributions are
  evaluated on technical merit, project scope, vendor neutrality, maintainability, and
  the contribution requirements in [CONTRIBUTING.md](CONTRIBUTING.md).
- **Code owners** review changes and help maintain the repository. The current code
  owners are listed in [`.github/CODEOWNERS`](.github/CODEOWNERS).
- **Project lead**: [Juraci Paixão Kröhling](https://github.com/jpkrohling), CEO of
  OllyGarden, is the project's sole governance authority.

Code ownership is an operational review responsibility; it does not grant governance
authority.

## Decision making

Maintainers seek consensus through public issues and pull requests whenever practical.
The project lead has final authority over project scope, governance, maintainer and code
owner appointments, releases, and resolution of disputes that cannot be settled by
consensus.

Governance decisions that materially affect contributors will be documented publicly in
the repository. Decisions involving reports under the Code of Conduct or undisclosed
security vulnerabilities may remain private to protect the people or systems involved.

## Changes to governance

Anyone may propose a governance change through a pull request. Governance changes require
approval from the project lead.
